2.change事件和keyup事件的区别
            keyup是键盘内容按下时触发
            change是输入框内容改变时触发
            比如：按下ctrl键就不会触发change，因为内容没有改变，但是会触发keyup
            按下a在放下则会触发change和keyup事件
